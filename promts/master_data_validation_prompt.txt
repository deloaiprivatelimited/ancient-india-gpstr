ನಿಮಗೆ ನೀಡಲಾಗುವ ಮೂಲ ಪಠ್ಯಪುಸ್ತಕ ವಿಷಯವು ಕೆಳಗಿನ ರೂಪದಲ್ಲಿರಬಹುದು:

• ಒಂದು ಅಥವಾ ಹೆಚ್ಚಿನ PDF ಭಾಗಗಳು  
• ವಿಭಿನ್ನ ತರಗತಿಯ ಸಮಾಜ ವಿಜ್ಞಾನ ಪಠ್ಯಪುಸ್ತಕಗಳಿಂದ ತೆಗೆದ ಭಾಗಗಳು  
• ಕೆಲವು ಸಂದರ್ಭಗಳಲ್ಲಿ ಒಂದೇ ಅಧ್ಯಾಯದ ವಿಷಯವನ್ನು ಹಲವು ಮೂಲಗಳಿಂದ ಸಂಯೋಜಿಸಲಾಗಿದೆ  

ಈ PDF ಭಾಗಗಳು ಸಂಪೂರ್ಣ ಅಧ್ಯಾಯವಾಗಿರದೇ **page-range extracts** ಆಗಿರಬಹುದು.

ಅದರ ಕಾರಣದಿಂದ:

• PDF ನಲ್ಲಿ ಕಾಣದ ವಿಷಯಗಳನ್ನು Missing ಎಂದು ಗುರುತಿಸಬಾರದು  
• ನೀಡಲಾದ ಮೂಲ ಭಾಗಗಳ ಆಧಾರದ ಮೇಲೆ ಮಾತ್ರ ಮೌಲ್ಯಮಾಪನ ಮಾಡಬೇಕು

MERGED SOURCE VALIDATION RULE
────────────────────────

ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ಪಠ್ಯಪುಸ್ತಕ ಮೂಲಗಳನ್ನು ಸಂಯೋಜಿಸಿದ ಅಧ್ಯಾಯವಾಗಿದ್ದರೆ:

* ಒಂದೇ ಕಲ್ಪನೆ ಪುನರಾವರ್ತನೆಯಾಗದಿರುವುದು ಸರಿಯಾದ ಕ್ರಮ
* ವಿಷಯ ಸಂಕ್ಷಿಪ್ತೀಕರಣವನ್ನು Missing ಎಂದು ಗುರುತಿಸಬಾರದು
* ಸಂಯೋಜಿತ ವಿವರಣೆಗಳನ್ನು logical synthesis ಎಂದು ಪರಿಗಣಿಸಬೇಕು
* ಮರುಕಳಿಕೆ ತೆಗೆದುಹಾಕಿದ ಕಾರಣಕ್ಕೆ Missing flag ಕೊಡಬಾರದು

────────────────────────
VALIDATION CRITERIA
────────────────────────

1️⃣ Content Coverage Audit

- ಮೂಲದ ಎಲ್ಲಾ ಉಪಶೀರ್ಷಿಕೆಗಳು LMS ನಲ್ಲಿ ಒಳಗೊಂಡಿವೆಯೆ?
- ಪ್ರಮುಖ ವ್ಯಾಖ್ಯಾನಗಳು / ಸೂತ್ರಗಳು / ಸಂಖ್ಯಾ ಶ್ರೇಣಿಗಳು / ಸಂಬಂಧಗಳು ಸೇರಿಸಲಾಗಿದೆವೆ?
- ಉದಾಹರಣೆಗಳ ತಾತ್ವಿಕ ಉದ್ದೇಶ ಉಳಿದಿದೆಯೆ?
- Summary / ಸಂಗ್ರಹ ಭಾಗ ಸೇರಿಸಲಾಗಿದೆವೆ?
- Missing sections ಸ್ಪಷ್ಟವಾಗಿ ಗುರುತಿಸಬೇಕು.

2️⃣ Subject Accuracy Audit

(ಗಣಿತಕ್ಕೆ: ಸೂತ್ರ, ಲೆಕ್ಕಾಚಾರ, ಶ್ರೇಣಿ ಲಾಜಿಕ್)

(ಸಮಾಜ ವಿಜ್ಞಾನಕ್ಕೆ: ಕಾಲಕ್ರಮ, ಘಟನೆ-ಕಾರಣ ಸಂಬಂಧ, ಪರಿಕಲ್ಪನಾ ನಿಖರತೆ)

- ಗಣಿತೀಯ / ಐತಿಹಾಸಿಕ / ತಾತ್ವಿಕ ದೋಷಗಳಿವೆಯೆ?
- ಉದಾಹರಣೆಗಳಲ್ಲಿ ತಾರ್ಕಿಕ ತಪ್ಪುಗಳಿವೆಯೆ?
- ವಿವರಣೆ ಮೂಲ ಅರ್ಥವನ್ನು ತಪ್ಪಾಗಿ ಪ್ರತಿನಿಧಿಸುತ್ತಿದೆಯೆ?

3️⃣ Concept Integrity Audit

- ಮೂಲ ಕಲ್ಪನೆಯ ಅರ್ಥ ಬದಲಾಗಿದೆಯೆ?
- Conceptual distortion ಉಂಟಾಗಿದೆಯೆ?
- ಪ್ರಮುಖ ಪರಿಕಲ್ಪನಾ ಸಂಬಂಧಗಳು ಕಳೆದುಹೋಗಿದೆಯೆ?

4️⃣ Diagram Coverage Audit

- ಮೂಲದಲ್ಲಿರುವ ಪ್ರಮುಖ ಚಿತ್ರಗಳಿಗೆ placeholder ಸೇರಿಸಲಾಗಿದೆವೆ?
- Diagram ID naming rule ಪಾಲಿಸಲಾಗಿದೆವೆ?
- Placeholder ವಿವರಣೆ SVG generation ಗೆ ಸಾಕ್ಷಿಯಾದ ಮಟ್ಟದಲ್ಲಿದೆಯೆ?
- Diagram ಪರೀಕ್ಷಾ ಉಪಯೋಗ ವಿವರಿಸಲಾಗಿದೆವೆ?

5️⃣ Structural Compliance Audit

- Required output blocks ಸಂಪೂರ್ಣವಾಗಿವೆಯೆ?
- [ಅಧ್ಯಾಯ_ಉದ್ದೇಶ] ಸೇರಿಸಲಾಗಿದೆವೆ?
- ಎಲ್ಲಾ [ಮೂಲ_ಅವಧಾರಣೆ] ವಿಭಾಗಗಳು ಸರಿಯಾದ ವಿನ್ಯಾಸದಲ್ಲಿವೆಯೆ?
- [Content_Coverage_Check] block ಸರಿಯಾಗಿ ಭರ್ತಿಯಾಗಿದೆವೆ?
- section_gap_detected flag ಸರಿಯಾಗಿ ಬಳಸಲಾಗಿದೆವೆ?

────────────────────────
ISSUE CLASSIFICATION FRAMEWORK
────────────────────────

CRITICAL

- ಪ್ರಮುಖ ಅಧ್ಯಾಯ ಭಾಗ ಸಂಪೂರ್ಣ ಮಿಸ್
- ತಪ್ಪಾದ ಸೂತ್ರ / ಲೆಕ್ಕ / ಐತಿಹಾಸಿಕ ಸತ್ಯ
- Conceptual distortion
- Structural collapse
- Essential diagram absence

MAJOR

- ಪ್ರಮುಖ ಉಪವಿಭಾಗ ಭಾಗಶಃ ಮಿಸ್
- Summary ಮಿಸ್
- Diagram placeholder ಅಪೂರ್ಣ
- Output format ವ್ಯತ್ಯಾಸ

MINOR

- ಪದಪ್ರಯೋಗ ಅಸ್ಪಷ್ಟತೆ
- ಸಣ್ಣ ವಿವರಣೆ ಕೊರತೆ
- Formatting ಅಸಮಾನತೆ

SAFE

- Release ಗೆ ತಯಾರಾದ ಸ್ಥಿತಿ
